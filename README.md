# Silent Walk Pro

**Лёгкая и быстрая утилита для Windows, написанная на Rust.**

Программа автоматизирует механику «тихого шага» (crouch-walking) в игре RUST.
Вместо того чтобы вручную спамить приседанием, вы просто зажимаете **W** (вперёд), а макрос сам периодически отправляет сигнал нажатия **Ctrl** (присед).
Утилита работает в фоновом режиме, прячет системный трей и использует `SendInput` (скан-коды), что делает её максимально незаметной и эффективной.

## ✨ Возможности

- **🎛 Два режима активации:**
  - `DoubleTap` — двойное нажатие клавиши-бинда в заданный промежуток времени.
  - `SinglePress` — простое одиночное нажатие для включения/выключения.

- **⌨️ Гибкая настройка клавиш:**
  - Выбор из выпадающего списка (буквы, цифры, F-клавиши, кнопки мыши).
  - Функция **Record** — нажмите любую клавишу, чтобы назначить её на бинд.

- **⏱ Тонкая настройка таймингов:**
  - Crouch Hold — длительность удержания приседа (20–500 мс).
  - Walk Delay — пауза между приседаниями (50–1000 мс).
  - Double Tap Window — окно времени для двойного нажатия (100–1000 мс).
  - Jitter — случайный разброс таймингов (0–50 мс) для предотвращения детекта.

- **🖥 Системный трей:**
  - Сворачивание/разворот окна по клику.
  - Контекстное меню (Show/Hide, Quit).

- **💾 Автосохранение** настроек в `%APPDATA%\silent_walk\config.json`.

- **📊 Мониторинг в реальном времени** — индикация состояния клавиш W и бинда.

## ⚙️ Как это работает

1. Фоновый поток (`Worker`) постоянно опрашивает состояние клавиш через `GetAsyncKeyState`.
2. При удержании **W** и активном макросе генерируются синтетические нажатия **Ctrl** с заданными задержками.
3. Используется Win32 API `SendInput` с флагом `KEYEVENTF_SCANCODE` — это работает на уровне скан-кодов и лучше обходит защиты в играх.
4. Графический интерфейс построен на **egui** и общается с воркером через `Arc<Mutex<...>>`.
