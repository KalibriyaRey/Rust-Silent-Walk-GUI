# Rust-Silent-Walk-GUI
Silent Walk Pro — это лёгкая и быстрая утилита для Windows, написанная на Rust. Программа автоматизирует механику «тихого шага» (crouch-walking) в игре RUST Вместо того чтобы вручную спамить приседанием, вы просто зажимаете W (вперёд), а макрос сам периодически отправляет сигнал нажатия Ctrl (присед).
